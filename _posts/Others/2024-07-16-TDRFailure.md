如果你遇到了蓝屏故障如果你不去管可能会经常出现蓝屏死机的情况。那么如何在Win10/11上修复这种错误。

```
终止代码：VIDEO_TDR_FAILURE
失败的操作：nviddmkm.sys
```

Windows 10/11 中Video TDR 失败的原因

- Problem with graphics card
- Outdated graphics drivers
- Background processes
- Overheating component

# 解决方案 1：扫描并修复损坏的系统文件

有时这些文件可能会损坏或被不正确的替换

有一个简单的方法可以解决

在搜索栏中输入CMD以管理员身份运行并输入命令

```
sfc/scannow
```

然后按下回车键，全系统扫描现在将认真搜索并替换任何损坏的文件可能需要几分钟，扫描完成后重新启动电脑，并检查VIDEO_TDR_FAILURE是否停止

如果你发现自己仍然面临这种故障

# 解决方案 2：执行启动修复

> 我通过这个解决了

第一次之后的视频 TDR 故障解决方案：

很可能是由于启动程序无法正确加载

按照以下简单步骤执行Windows 10 或 11 上的启动修复

在设置菜单并点击更新和安全，现在导航到恢复部分，在高级启动选项卡下单击，现在重新开始，你可能会被提示输入你的帐户密码如果是，则继续

完成这些后再做这些步骤，启动修复将自动检测并修复

# 解决方案 3：更新显卡和芯片组驱动程序

过时的显示驱动程序可能导致视频 TDR 故障错误

确保您的显卡驱动程序，按照以下简单的步骤即可获得最新信息

按下 Windows 和 X 键并选择设备管理器，查找显示驱动程序并单击，旁边的箭头可展开选项，然后右键单击特定驱动程序，你的系统现在将开始一项任务，找到最合适的驱动并自动安装它们

重新启动并查看视频 TDR 是否故障

# 解决方案 4：在安全模式下重新安装驱动程序

问题按下 Windows 和 R 键，启动运行窗口，然后在窗口中输入 `msconfig` 并按回车键打开系统配置

检测启动，准备重新启动

系统启动进入安全模式后，按 Windows 和 X 键打开设备管理器，右键单击有问题的驱动程序，并选择立即卸载，访问制造商网站并下载最适合您的驱动程序版本
