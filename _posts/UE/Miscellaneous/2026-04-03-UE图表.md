# 0、前言

在UE中想要实现一个好看可交互的图表实现起来非常的麻烦，但是有一种办法可以将网页在UE的UI中显示，借助这个办法就可以配合知名度很高的ECharts实现直观，交互丰富，可高度个性化定制的数据可视化图表。

> ECharts网页：[Apache ECharts](https://echarts.apache.org/zh/index.html)

那么接下来我要做到的效果是

1. 在UE中显示简单折线图
2. 按键传递UE产生的数据给html，然后此UI同步数据更新折线图
3. 与html交互，产生的数据发给UE并log打印出来

# 1、试用ECharts

先快速入门一下，然后再在UE中使用。

## 下载JS库

> 下载方式：[下载](https://echarts.apache.org/zh/download.html)

我使用的是[在线定制](https://echarts.apache.org/zh/builder.html)，我把我需要的勾上（chart: Bar、Line、Radar, coordinate systems: Grid、Polar, 默认, others: 只勾选工具集）然后下载。

将下载好的js库放入js文件夹
然后新建一个html

```html
<!DOCTYPE html>
<html lang="en" style="height: 100%">
<head>
 <meta charset ="utf-8">
</head>
<body style="height: 100%; margin: 0">
  <!--容器，后面脚本会根据此容器id放入echart-->
  <div id="container" style="height: 100%"></div>
  
  <!--引入js库-->
  <script src='js/echarts.js'></script>
  <script type="text/javascript">
    //初始化ECharts实例
    var dom = document.getElementById('container');
    var myChart = echarts.init(dom);
    // ECharts基础配置:折线图
    var option;
    option = {
      xAxis: {
        type: 'category',
        data: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
      },
      yAxis: {
        type: 'value'
      },
      series: [
        {
          data: [150, 230, 224, 218, 135, 147, 260],
          type: 'line'
        }
      ]
    };

	//显示
    myChart.setOption(option);

    // 窗口大小自适应
    window.onresize = function () {
      myChart.resize();
    };
  </script>
</body>
</html>
```

双击后即可看到一条普通的折线图，此示例参考自[简单折线图](https://echarts.apache.org/examples/zh/editor.html?c=line-simple)，使用起来非常简单，还有很生动且完善的文档，比如快速上手的[使用手册](https://echarts.apache.org/handbook/zh/get-started/)，所有的[配置项](https://echarts.apache.org/zh/option.html#title)和很多[示例](https://echarts.apache.org/examples/zh/index.html)

# 2、接入UE

刚才用过ECharts之后，很明显，他只是一个方便在网页写图表的库和UE没有一点关系，不过之前也说了，有一种办法可以将网页显示在UI上，这个工具我用的是[Web UI](https://tracerinteractive.com/plugins/webui)，他提供免费下载渠道，功能是在UI中创建基于网页的UI并且JavaScript与UE相互通信等。

下载好以后会有两个文件夹：`WebUI`和`JsonLibrary`，将这两个文件夹放到引擎插件的`Marketplace`下，打开项目后开启这俩插件，创建Widget Blueprint就会发现Common栏多了一个`Web Interface`，拖进来后如图所示填写`Web Interface`实例的`LoadFile`函数的参数，文件名是你创建的，现在将他`Add to viewport`就已经可以看到你上一章节创建的折线图了

![](..\..\..\styles\images\Miscellaneous\ECharts\LoadFile.png)

当然，现阶段UE与这个UI的交互仅仅是鼠标和html的交互，这远远达不到想要的效果

## 挂载函数，绑定委托

要想进行与UE的交互还需要在html里写一些东西，将代码放到第一章写的script下方，不然`myChart`这个变量还没初始化

```html
<script>
  // ===================== 必加：UE WebUI 通信助手函数 =====================
  if (typeof ue != "object" || typeof ue.interface != "object") {
    if (typeof ue != "object") ue = {};
    ue.interface = {};
    ue.interface.broadcast = function (name, data) {
      if (typeof name != "string") return;
      var args = [name, ""];
      if (typeof data != "undefined") args[1] = data;
      var hash = encodeURIComponent(JSON.stringify(args));
      if (
        typeof history == "object" &&
        typeof history.pushState == "function"
      ) {
        history.pushState({}, "", "#" + hash);
        history.pushState({}, "", "#" + encodeURIComponent("[]"));
      } else {
        document.location.hash = hash;
        document.location.hash = encodeURIComponent("[]");
      }
    };
  }
  // 全局ue5函数，简化调用：ue5(事件名, 数据)
  var ue5 = ue.interface.broadcast;
</script>
<script>
  // ===================== 供UE调用的【数据更新函数】（UE→ECharts） =====================
  // 挂载到ue.interface对象上，UE可通过Call节点调用此函数，参数为JSON格式的图表数据
  ue.interface.updateEChartsData = function (chartData) {
    // chartData是UE传递的JSON对象，直接更新ECharts配置
    myChart.setOption({
      series: [
        {
          data: chartData[0],type: 'line'
        }
      ],
    });
      
    // 可选：将更新结果回传给UE（前端→UE）
    var res = { status: chartData, time: new Date().getTime() };
    ue5("ECharts_Updated", JSON.stringify(res));
  };
    
  // ===================== ECharts交互事件→UE 数据发送（核心：ECharts→UE） =====================
  // 点击ECharts图表，将点击的节点数据发送给UE
  myChart.on("click", function (params) {
    // 构造要发送给UE的JSON数据
    var clickData = {
      chartType: "line",
      seriesName: params.seriesName,
      dataIndex: params.dataIndex,
      value: params.value,
      name: params.name,
    };
    // 通过ue5()发送给UE，事件名：ECharts_Click（UE端需监听此名称）
    ue5("ECharts_Click", JSON.stringify(clickData));
  });
</script>
```

JavaScript代码很简单，注释也写清楚了，接下来就是UE蓝图了

### ECharts到UE

![](..\..\..\styles\images\Miscellaneous\ECharts\BindEvent.png)

绑定html回调函数，就是JavaScript里的`ue5()`，Name就是第一个参数，Data就是第二个参数,`Stringfy`函数是`JsonLibrary`里的

然后你会发现还有一个Callback没有用到这个东西我看文档是说`ue5()`这个函数可以接收四个参数，但是我上方写的UE WebUI 通信助手函数只能接收两个，所以需要更改助手函数，下文所示，这下就可以了

```html
<script>
  //==================UE WebUI 通信助手函数=====================
  ("object" != typeof ue && (ue = {}),
    (uuidv4 = function () {
      return "10000000-1000-4000-8000-100000000000".replace(
        /[018]/g,
        function (t) {
          return (
            t ^
            (crypto.getRandomValues(new Uint8Array(1))[0] & (15 >> (t / 4)))
          ).toString(16);
        },
      );
    }),
    (ue5 = (function (r) {
      return "object" != typeof ue.interface ||
        "function" != typeof ue.interface.broadcast
        ? ((ue.interface = {}),
          function (t, e, n, o) {
            var u, i;
            "string" == typeof t &&
              ("function" == typeof e && ((o = n), (n = e), (e = null)),
              (u = [t, "", r(n, o)]),
              void 0 !== e && (u[1] = e),
              (i = encodeURIComponent(JSON.stringify(u))),
              "object" == typeof history &&
              "function" == typeof history.pushState
                ? (history.pushState({}, "", "#" + i),
                  history.pushState({}, "", "#" + encodeURIComponent("[]")))
                : ((document.location.hash = i),
                  (document.location.hash = encodeURIComponent("[]"))));
          })
        : ((i = ue.interface),
          (ue.interface = {}),
          function (t, e, n, o) {
            var u;
            "string" == typeof t &&
              ("function" == typeof e && ((o = n), (n = e), (e = null)),
              (u = r(n, o)),
              void 0 !== e
                ? i.broadcast(t, JSON.stringify(e), u)
                : i.broadcast(t, "", u));
          });
      var i;
    })(function (t, e) {
      if ("function" != typeof t) return "";
      var n = uuidv4();
      return (
        (ue.interface[n] = t),
        setTimeout(
          function () {
            delete ue.interface[n];
          },
          1e3 * Math.max(1, parseInt(e) || 0),
        ),
        n
      );
    })));
</script>
```

第三个参数传函数引用，第四个参数传销毁第三个参数的函数引用的时间默认是1秒，我真是实验下来发现，默认1秒，在UE中延迟2秒后调用`Callback`发现`Callback`的`Valid`仍然返回true，但是函数已经无法执行，这一点需要注意

```javascript
ue5("ECharts_Click", JSON.stringify(clickData), function(v){
  myChart.setOption({
  series: [
    {
      data: [4,5,v],
      type: "line",
    },
  ],
});
}, 3);//3秒
```

![](..\..\..\styles\images\Miscellaneous\ECharts\CallECharts.png)
